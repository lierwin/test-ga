# Meteor 3 + Blaze

This is a simple example of how to use Blaze with Meteor 3.

## How to use

1. Clone this repo
2. Run `meteor npm install`
3. Run `meteor`
4. Open `http://localhost:3000` in your browser

## Libraries used

- [Blaze](https://www.blazejs.org/)
- [Meteor](https://www.meteor.com/)
